## Problem description

There are 5 colored pegs with the following colors: red, green, blue, pink, yellow. The pegs are all named in the format <color>_peg. Additionally there are 4 colored rings: red, green, blue and yellow. Similarly to the pegs, the rings are named <color>_ring.

The robot arm starts out in a default position, while the rings start in the following positions:
- red_ring - pink_peg
- green_ring - green_peg
- blue_ring - blue_peg
- yellow_ring - red_peg

The goal is to transfer all rings to the pegs of their color.
