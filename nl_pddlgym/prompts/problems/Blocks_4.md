## Problem description

There are 5 given blocks (a,b,c,d,e,f) on the table and one available robot (robot). The blocks are stacked on top of each other in the following order (starting from the bottom): f,e,d,c,b. The robot is currently holding the remaining block a. The goal is to stack the blocks in the following order (starting from the bottom): a,f,e,d,c,b
