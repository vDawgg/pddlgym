## Problem description

There are 5 pegs (peg1 - peg5) and 7 rings (d1 - d7). The rings have the following order according to their size (largest to smallest): d7, d6, d5, d4, d3, d2, d1. peg1 has rings stacked on it in the following order (bottom to top): d7, d6, d5. peg2 has rings stacked on it in the following order (bottom to top): d4. peg3 has no rings stacked on it. peg4 has rings stacked on it in the following order (bottom to top): d3. peg5 has rings stacked on it in the following order (bottom to top): d2, d1. The goal is to stack the rings on peg5 from largest to smallest.
