## Problem description

There are 23 given blocks (b0 - b22). There are 6 stacks of blocks on the table (given in the order from top to bottom) b7-b8-b9-b10-b11, b12-b13-b14, b15-b16-b17-b18, b0-b1-b2, b19-b20-b21-b22, b3-b4-b5-b6. The goal is to rearrange the blocks in the following stacks: b7-b9-b11-b0, b22-b2-b1-b17-b10, b14-b12-b16-b13, b21-b6-b3-b15, b5-b20-b8-b4.
