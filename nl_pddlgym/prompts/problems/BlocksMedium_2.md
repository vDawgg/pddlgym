## Problem description

There are 23 given blocks (b0 - b22). There are 6 stacks of blocks on the table (given in the order from top to bottom) b8-b9-b10-b11, b12-b13-b14-b15, b16-b17-b18-b19, b0-b1-b2, b20-b21-b22, b3-b4-b5-b6-b7. The goal is to rearrange the blocks in the following stacks: b4-b22-b11, b17-b18-b15-b14, b19-b10-b3-b20, b16-b1-b6-b21.
