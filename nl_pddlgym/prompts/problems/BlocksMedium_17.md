## Problem description

There are 19 given blocks (b0 - b18). There are 5 stacks of blocks on the table (given in the order from top to bottom) b8-b9-b10-b11, b12-b13-b14-b15, b16-b17-b18, b0-b1-b2-b3, b4-b5-b6-b7. The goal is to rearrange the blocks in the following stacks: b1-b16-b11-b7-b10, b3-b18-b12, b13-b9-b4-b17-b2, b0-b8-b6.
