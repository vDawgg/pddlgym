## Problem description

There are 4 colored pegs with the following colors: blue, green, pink and red. The pegs are all named in the format <color>_peg. Additionally there are 3 colored rings: blue, green and red. Similarly to the pegs, the rings are named <color>_ring.

The robot arm starts out in a default position, while the rings start in the following positions:
- red_ring - green_peg
- green_ring - blue_peg
- blue_ring - red_peg

The goal is to transfer all rings to the pegs of their color.
