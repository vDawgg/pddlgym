## Problem description

There are 14 given blocks (b0 - b13). There are 4 stacks of blocks on the table (given in the order from top to bottom) b10-b11-b12-b13, b0-b1-b2, b3-b4-b5, b6-b7-b8-b9. The goal is to rearrange the blocks in the following stacks: b9-b8-b1, b6-b3-b5-b2-b10, b12-b13-b11-b0-b4.
