## Problem description

There are 4 pegs (peg1 - peg4) and 5 rings (d1 - d5). The rings have the following order according to their size (largest to smallest): d5, d4, d3, d2, d1. peg1 has rings stacked on it in the following order (bottom to top): d5, d4. peg2 has rings stacked on it in the following order (bottom to top): d3. peg3 has rings stacked on it in the following order (bottom to top): d2, d1. peg4 has no rings stacked on it. The goal is to stack the rings on peg4 from largest to smallest.
