## Problem description

There are 3 pegs (peg1 - peg3) and 3 rings (d1 - d3). The rings have the following order according to their size (largest to smallest): d3,d2,d1. peg2 and peg3 have no rings stacked on them. peg1 has rings stacked on it in the following order (bottom to top): d3,d2,d1. The rings should be stacked from largest to smallest on peg p3.
