## Problem description

There are 3 pegs (peg1, peg2 and peg3) and 3 rings (d1, d2 and d3). The rings have the following order according to their size (largest to smallest): d3, d2, d1. peg1 has rings stacked on it in the following order (bottom to top): d2. peg2 has rings stacked on it in the following order (bottom to top): d3. peg3 has rings stacked on it in the following order (bottom to top): d1. The goal is to stack the rings on peg3 from largest to smallest.
