## Problem description

There are 21 given blocks (b0 - b20). There are 5 stacks of blocks on the table (given in the order from top to bottom) b10-b11-b12, b13-b14-b15-b16, b17-b18-b19-b20, b0-b1-b2-b3-b4, b5-b6-b7-b8-b9. The goal is to rearrange the blocks in the following stacks: b13-b16-b0-b1, b14-b2-b17-b19-b10, b7-b3-b8-b12-b11, b6-b20, b18-b4-b15-b5-b9.
