## Problem description

There are 17 given blocks (b0 - b16). There are 4 stacks of blocks on the table (given in the order from top to bottom) b8-b9-b10-b11, b12-b13-b14-b15-b16, b0-b1-b2, b3-b4-b5-b6-b7. The goal is to rearrange the blocks in the following stacks: b8-b16-b2-b3-b1, b12-b0-b10, b11-b9-b13-b4-b15, b7-b14-b6-b5.
