## Problem description

There are five orders that need to be fulfilled (o1 - o5), six possible types of products (p1 - p5) and a maximum of six possible stacks (n0 - n5). There are no open stacks currently available. The machine is currently available and not configured for any specific product type. None of the orders have started to be processed. Order o1 includes a products of type p1 and p2, o2 of p3 and p4, o3 of p1 and p5, o4 of p2 and p5, o5 of p3 and p4. The goal is for all orders to be shipped.
