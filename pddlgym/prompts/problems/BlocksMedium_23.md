## Problem description

There are 23 given blocks (b0 - b22). There are 6 stacks of blocks on the table (given in the order from top to bottom) b9-b10-b11, b12-b13-b14, b15-b16-b17-b18, b19-b20-b21-b22, b0-b1-b2-b3-b4, b5-b6-b7-b8. The goal is to rearrange the blocks in the following stacks: b7-b10-b11, b8-b9-b22-b18-b12, b16-b13-b6-b14, b3-b15-b21-b19, b4-b0-b20, b1-b17-b2-b5.
