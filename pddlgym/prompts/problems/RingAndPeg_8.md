## Problem description

There are 5 colored pegs with the following colors: green, orange, purple, red and yellow. The pegs are all named in the format <color>_peg. Additionally there are 3 colored rings: orange, purple and yellow. Similarly to the pegs, the rings are named <color>_ring.

The robot arm starts out in a default position, while the rings start in the following positions:
- yellow_ring - yellow_peg
- orange_ring - purple_peg
- purple_ring - orange_peg

The goal is to transfer all rings to the pegs of their color.
