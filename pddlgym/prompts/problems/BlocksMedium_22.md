## Problem description

There are 21 given blocks (b0 - b20). There are 5 stacks of blocks on the table (given in the order from top to bottom) b8-b9-b10-b11-b12, b13-b14-b15-b16-b17, b18-b19-b20, b0-b1-b2-b3, b4-b5-b6-b7. The goal is to rearrange the blocks in the following stacks: b1-b11-b14-b17, b5-b3-b19, b6-b7-b16-b4-b9.
