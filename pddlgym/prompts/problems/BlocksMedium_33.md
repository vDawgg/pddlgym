## Problem description

There are 21 given blocks (b0 - b20). There are 5 stacks of blocks on the table (given in the order from top to bottom) b8-b9-b10-b11-b12, b13-b14-b15-b16, b17-b18-b19-b20, b0-b1-b2-b3, b4-b5-b6-b7. The goal is to rearrange the blocks in the following stacks: b10-b16-b12-b17, b13-b14-b11-b5-b18, b1-b4-b0-b3-b20, b8-b9-b19-b6, b15-b2-b7.
