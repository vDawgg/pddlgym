## Problem description

There are 21 given blocks (b0 - b20). There are 6 stacks of blocks on the table (given in the order from top to bottom) b7-b8-b9-b10, b11-b12-b13, b14-b15-b16, b17-b18-b19-b20, b0-b1-b2-b3, b4-b5-b6. The goal is to rearrange the blocks in the following stacks: b17-b9-b8-b0, b20-b5-b10, b1-b7-b14-b6-b11, b2-b4-b18-b16-b12, b15-b3-b19-b13.
