## Problem description

There are 16 given blocks (b0 - b15). There are 4 stacks of blocks on the table (given in the order from top to bottom) b8-b9-b10-b11, b12-b13-b14-b15, b0-b1-b2-b3-b4, b5-b6-b7. The goal is to rearrange the blocks in the following stacks: b8-b14-b7-b0, b15-b11-b5-b1, b4-b13-b2-b3, b12-b10-b6-b9.
