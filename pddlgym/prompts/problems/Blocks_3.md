## Problem description

There are 5 given blocks (a,b,c,d,e,f) on the table and one available robot (robot). The blocks are stacked on top of each other in the following order (starting from the bottom): c,b,a with e and f both on the table and no other blocks stacked on top of them. The robot is currently holding the remaining block d. The goal is to stack the blocks in the following order (starting from the bottom): f,e,d,c,b,a
