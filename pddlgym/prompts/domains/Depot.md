## Domain description

A truck is tasked with transferring a set of pallets between a set of distributors.

The available actions are:
- Drive
    - Drive a truck to a specific place
- Lift
    - Lift a crate using a hoist
- Drop
    - Drop a crate using a hoist
- Load
    - Load a crate from a hoist onto a truck
- Unload
    - Unload a crate from a truck onto a hoist
