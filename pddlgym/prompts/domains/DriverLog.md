## Domain description

A set of robots are tasked with driving a set of items from one place to another using a set of trucks.

The actions available to the robot are:
- load-truck
    - Loads an item on a truck.
- unload-truck
    - Unloads an item from a truck.
- board-truck
    - Boards a robot on a truck.
- disembark-truck
    - Disembarks a robot from a truck.
- drive-truck
    - Drives a truck to a specific location.
- walk
    - Walks a robot to a specific location.
